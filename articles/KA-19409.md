# [!UICONTROL 文件收集器] ACS中的活动替代项

## 描述

因为我们没有 [!UICONTROL 文件收集器] 活动时，您可以使用以下活动来构建它，并且您将获得大致相同的功能

## 分辨率


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



的 [!UICONTROL 文件收集器] 将允许从外部服务器下载基于文件名模式的多个文件。 每个文件都会传递到 [!UICONTROL 加载文件] 活动，向更新中提供数据。
完成后，将继续使用以下文件等。
文件传输还具有 [!UICONTROL 历史化设置]，如您从 [!UICONTROL 文件收集器].
