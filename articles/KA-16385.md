# 不适用于工作区中划分报表中的提升度和置信度

## 描述


获取包含的划分报表时出现不适用的原因 [!DNL Target] A4T报表中的体验(位于Analysis Workspace中)。


## 分辨率


提升度和置信度仅针对 [!UICONTROL Target] 体验维度。 当前不支持划分。 由于区段在报表引擎的不同级别发生，因此可以将其用作解决方法。
