# 使用at.js 2.x创建mbox

## 描述


`mboxCreate` 方法不适用于at.js 2.x如何在页面级别创建mbox div?

我尝试了下面的代码，但没有运气：


```
div class="mboxDefault"
 adobe.target.getOffer({
 "mbox": "customTestMbox",
 "params": {
 "status": "true"
 },
 "success": function(offer) {
 adobe.target.applyOffer( {
 "mbox": "customTestMbox",
 "offer": offer
 } );
 },
 "error": function(status, error) {
 console.log('Error', status, error);
 }
 });
 /div
```

## 分辨率


尝试在脚本之前关闭div标记。

样例:


```
div class="mboxDefault" 
  default content to replace by offer 
/div 
script 
  var el = document.currentScript.previousElementSibling;
  adobe.target.getOffer({
    mbox: "mboxName",
    params: {
      param1: "value1",
      param2: "value2"
    },
    success: function(offer) {
      adobe.target.applyOffer({
        mbox: "mboxName",
        selector: el,
        offer: offer
      });
    },
    error: function(error) {
      console.error(error);
      el.style.visibility = "visible";
    }
  });
/script
```
