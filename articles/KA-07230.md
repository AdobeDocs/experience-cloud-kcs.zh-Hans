# 当天日期的Data Warehouse请求能否返回当前数据？ | [!DNL Analytics]

## 描述


<b>环境</b>
Data Warehouse

<b>问题</b>
当天日期的Data Warehouse请求能否返回当前数据？


## 解决方法


<b>回答</b>
可以，您可以计划每小时Data Warehouse请求，以传递相同日期（当天）的每小时文件以返回当前数据。
