# data warehouse今天的日期请求是否可以返回当前数据？ | [!DNL Analytics]

## 描述


<b>环境</b>
data warehouse

<b>问题</b>
data warehouse今天的日期请求是否可以返回当前数据？


## 分辨率


<b>回答</b>
是，您可以安排每小时Data warehouse请求在同一日期（当天）提交每小时文件，以返回当前数据。
