# AEM：如何监视复制队列

## 描述 {#description}


### <b>环境</b>

- AEM 6.5
- Experience Manager


### <b>问题/症状</b>

当发布过程未按预期继续时，复制队列可能会卡住。

<b>注意</b>：在大多数情况下，当发布过程未按预期继续时，复制队列可能会卡住。 监视复制队列可以帮助您有效地注意到它安静运行。


## 解决方法 {#resolution}


通过访问 [以下URL](https://localhost:4502/etc/replication/agents.author/publish/jcr:content.queue.json).

<b>结果</b>：
