# JQuery 软件未更新至版本 3.6.2

## 描述 {#description}

<b>环境</b>
Experience Manager


<b>问题/症状</b>
JQuery 组件未更新到最新版本。


## 解决方法 {#resolution}


AEM 将安全补丁应用于 1.12.4，并且出于向后兼容性原因未计划将其移除。截至今天，我们的分支 (1.12.4-aem) 包含 CVEs CVE-2015-9251、CVE-2019-11358、CVE-2020-11022 和 CVE-2020-11023 的修复程序。
