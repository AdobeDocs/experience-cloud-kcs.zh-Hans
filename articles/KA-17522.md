# 如何设置 Adobe I/O：身份验证 – 一步一步

## 描述

<br>本文档将引导您了解如何创建能够运行的证书、密钥和令牌。<br><br><br><br><br>目标<br><br><br><br><br><br>
为了提供清晰、可重现的步骤来生成 Adobe I/O 持有者令牌并运行 API 调用 DMA 解决方案，例如 [!DNL Target]。


## 分辨率

<br><br>步骤<br><br><br><br><br><br>
<b>创建持有者令牌以运行 Adobe I/O API 调用的步骤：</b>

1. 生成私钥和公共证书
2. 在 Adobe I/O 控制台中创建集成
3. 生成 JWT 令牌
4. 以 JWT 令牌交换 Access Bearer 令牌
5. 用户 Access Bearer 令牌运行 API 调用


- <b>根据以下文档生成私钥和公共证书：</b>https://www.adobe.io/apis/cloudplatform/console/authentication/createcert.html


   在 Mac 上，终端中内置了以下命令。 

   在计算机上，您必须下载 [!DNL Cygwin]（或其他个人喜好的工具）并从命令行运行它。 以下是下载和安装 [!DNL Cygwin] 的步骤：

   1. 浏览 https://cygwin.com/install.html
   2. 下载并运行 `setup-x86_64.exe`

   <b>*注意</b>：您的主目录是：`C:\cygwin64\home\WINDOWSUSER`

   您可以在安装阶段搜索并安装其他软件包。 我建议安装与“`curl`”和“`ssh.`”*相关的所有软件


   以下是要运行的命令：
   <br> <br> <br>

   | `$ openssl req -nodes -text -x509 -newkey rsa:2048 -keyout secret.pem -out certificate.pem -days 356` |
   | --- |

<br> 

    &lt;b>注意&lt;/b>：它会询问您几个必须填写才能生成证书的问题，请参见下面的屏幕快照：
![rtaImage](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_0/step_par/image/rtaImage.png "rtaImage")<br> 

    创建文件后，使用以下命令将 `secret.key` 转换为 `secret.pem`：
<br> <br> <br>

    | `$ openssl pkcs8 -topk8 -inform PEM -outform DER -` `in` `secret.pem -nocrypt secret.key` |
     | --- |

<br>
