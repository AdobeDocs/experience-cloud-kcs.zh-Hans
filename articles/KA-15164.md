# Campaign Classic： [!UICONTROL 批次大小] 选项自 [!UICONTROL 更新数据] 已忽略活动

## 描述

<b>环境</b>
Campaign Classic


<b>问题/症状</b>
此 *[!UICONTROL 批次大小]* 未遵循更新数据工作流活动中的属性。




## 解决方法


此 [!UICONTROL 批次大小] 字段用于选择每个数据库事务要更新的集客过渡元素数量。 例如，如果将500设置为批处理大小，则将在单个数据库事务中更新处理的500条记录。
