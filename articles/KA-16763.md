# 创建特征以捕获变量c_evar1中的所有值


了解如何在Adobe Audience Manager中创建特征以捕获所有c_evar1中的传入值。

## 描述 {#description}


### <b>环境</b>

Audience Manager



### <b>问题/症状</b>

如何创建一个特征来捕获变量中的所有传入值 `c_evar1` 在Adobe Audience Manager (AAM)中？


## 解决方法 {#resolution}


这可以通过以下帮助来实现 *正则表达式。*

条件，例如 `(c_evar1 matchesregex "*.*$")` 会生成一个特征，用于捕获进入AAM的所有值。 这将像是 `c_evar1` 已存在。
