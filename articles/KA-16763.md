# 创建特征以捕获变量c_evar1中的所有值

## 描述

<b>环境</b>
Audience Manager
<b>问题/症状</b>
如何在Adobe Audience Manager(AAM)中创建特征以捕获所有传入值？


## 分辨率


这是在 *正则表达式*.

条件，例如 `(c_evar1 matchesregex "*.*$")` 会创建一个特征来捕获传入AAM的所有值。 这将起到 *如果存在c_evar1*.
