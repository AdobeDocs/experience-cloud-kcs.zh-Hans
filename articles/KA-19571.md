# 如何检查由Service Pack安装更新的节点

## 描述

安装AEM 6.5.12.0等Service Pack时，您可能希望了解要更新的节点。 本文介绍如何使用调试日志检查这些节点。

<b>环境</b>

AEM 6.5

## 分辨率

在安装Service Pack之前，请为 `org.apache.jackrabbit.vault.packaging.impl.ActivityLog`. 调试日志会输出更新的节点路径。

1. 转到Web控制台Sling日志支持(http://localhost:4502/system/console/slinglog)
1. 选择“添加新日志记录器”按钮
1. 填写以下值，然后选择“保存”按钮

   示例配置:

   ```
   loglevel = DEBUG
   logfile = logs/activity.log
   logger = org.apache.jackrabbit.vault.packaging.impl.ActivityLog
   ```

1. 安装Service Pack
1. 检查调试日志
