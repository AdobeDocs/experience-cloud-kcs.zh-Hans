# Campaign Standard:活动用户档案在中未显示最新 [!UICONTROL 控制面板]

## 描述


据报告，活动用户档案在 [!UICONTROL 控制面板].

要查看此内容，请导航至 [!UICONTROL 控制面板] >性能监控>活动配置文件

请注意，活动用户档案计数不是最新的，直到14/1/2022显示的数据此后不显示任何数据/更新。


## 分辨率


要解决此问题，请检查 [!UICONTROL 帐单] 工作流，以确保活动联系人已更新。 如果工作流中存在任何错误，请解决这些错误并重新启动工作流。

[!DNL Campaign] 连续从内部系统中近乎实时地获取数据并将其存储在 [!UICONTROL 控制面板] 以所需的格式显示。 已失效的数据将不会显示在 [!UICONTROL 控制面板].
