# 在生成步骤期间，管道部署失败

## 描述 {#description}


<b>描述</b> :由于ui.frontend代码中的错误，在生成步骤期间管道执行失败

在build.log文件中，您会看到如下类型的错误消息：

## 解决方法 {#resolution}


- 在git存储库的根中找到文件pom.xml，并查找节点版本，例如：nodeVersionv10.13.0/nodeVersion

- 找到ui.frontend/package.json并添加已存在的devDependencies对象：例如：


```
"devDependencies": {"@types/node": "10.13.0",...}
```


注释：
1. 不得复制来自nodeVersionv10.13.0的v。
2. 在您的用例中，节点版本可能不同。 例如：12.22.7
