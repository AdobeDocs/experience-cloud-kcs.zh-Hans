# 片段列表未显示在 [!UICONTROL 电子邮件设计工具]

## 描述

问题语句 — 使用新 [!UICONTROL 电子邮件设计工具]，则只有OOTB片段会显示在片段列表中，而在尝试刷新列表时，列表会永远加载。

## 分辨率


当使用旧版编辑器创建片段时，可能会发生这种情况。 打开片段并通过新编辑器保存该片段将更新片段，之后列表将可正常使用。
