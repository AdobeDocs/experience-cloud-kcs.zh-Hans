<b>环境</b>
AEM 6.2与Brand Portal

<b>原因</b>
原因是AEM实例和Brand Portal云之间的时间差。
此差异会导致JWT过期。

<b>指定的消息</b>


```
Failed to get an access token 
JWT has expired
```



## 分辨率


<b>分辨率</b>

1. 确认AEM的OS时间。
2. 将时间更改为有效时间或使用ntp。
