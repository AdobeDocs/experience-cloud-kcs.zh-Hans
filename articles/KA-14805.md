# [!UICONTROL 收件箱呈现] 选项卡在安装IR包后不可见

## 描述

环境<br>
Campaign Classic
<br>问题/症状<br>
[!UICONTROL 收件箱呈现] 已安装并配置包。 配置后，将提交IR请求，并发送校样。 但是，打开校样时，缺少用于查看渲染的IR选项卡。


## 分辨率


检查 ``nmsDelivery`` 表单，其条件是 *[!UICONTROL 电子邮件投放能力]* *需要包* ，以便显示“IR”选项卡。 要修复此问题，请安装 *[!UICONTROL 电子邮件投放能力] 包。*
