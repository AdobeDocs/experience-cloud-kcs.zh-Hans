# Dispatcher更改未反映在INT环境中

## 描述 {#description}

<b>环境</b>
Experience ManagerCloud Manager


<b>问题/症状</b>
为了允许在发布者中访问逗号分隔值(CSV)、JavaScript对象表示法(JSON)和可扩展标记语言(XML)文件，我们在 `filters.any` （对于调度程序模块）。 部署后，更改会在开发环境中正常显示，但在INT上则不会。


## 解决方案 {#resolution}

在这种情况下，请确保将Web层配置添加到完整堆栈代码部署管道中。
