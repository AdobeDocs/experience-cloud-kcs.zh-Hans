# 无法保存相似模型

## 描述 {#description}

<b>环境</b>
Audience Manager


<b>问题/症状</b><br>在 AAM 中使用相似模型时，无法进行保存。<br>

## 解决方法 {#resolution}


如果相关模型使用来自 Marketplace 的数据，请确保已启用您在模型中使用的数据以进行建模。否则，相似模型将不会保存。如果不是这个问题，请打开支持案例。
