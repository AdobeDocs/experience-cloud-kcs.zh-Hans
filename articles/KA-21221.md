# 无法保存相似模型


了解如何解决Adobe Audience Manager (AAM)中的相似模型无法保存的问题。

## 描述 {#description}


### 环境

Audience Manager

### <b>问题/症状</b>

在AAM中使用相似模型时，无法进行保存。


## 解决方法 {#resolution}


如果相关模型使用来自 Marketplace 的数据，请确保已启用您在模型中使用的数据以进行建模。否则，相似模型将不会保存。如果不是这个问题，请打开支持案例。
