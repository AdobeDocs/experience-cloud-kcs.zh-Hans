# mod_security：无法上传大文件/资产


本文旨在通过更新到mod_security的最新版本来解决必须上载大文件/资产的问题。

## 描述 {#description}


<b>环境</b>

Adobe Experience Manager

<b>问题/症状</b>

mod_security的版本9无法上载大文件/资产。 需要通过导航到作者（绕过Dispatcher）来上传它们。
