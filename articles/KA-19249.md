# 如何以JSON格式导出资产的本地引用

## 描述


您可以在属性的引用选项卡中看到资产的本地引用。 （在“资产”控制台中，选择一个资产，从工具栏中选择属性，然后选择引用选项卡。） 使用AEM外的信息时，可以将其导出为JSON格式。

<b>环境</b>

AEMas a Cloud Service, AEM 6.5


## 分辨率


访问以下URL“1”将返回JSON格式的本地引用。 响应中的“路径”值是引用目标资产的内容路径。

“1”http://localhost:4502/bin/wcm/references.json?path=path用于定位资产

<b>用法示例：</b>
http://localhost:4502/bin/wcm/references.json?path=/content/dam/wknd/en/activities/hiking/equipment_6.jpg
