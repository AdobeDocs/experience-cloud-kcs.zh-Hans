# 是 [!DNL Target] 配置文件属性被每次访问覆盖？

## 描述


目标配置文件属性值是否每次都被覆盖？


## 分辨率


否，默认情况下不会覆盖配置文件属性。 除非您明确更改该值，否则在用户档案过期之前，该值将保持不变。

您可以通过查看mbox跟踪中的配置文件属性值来验证这一点，以观察值之前和之后。
