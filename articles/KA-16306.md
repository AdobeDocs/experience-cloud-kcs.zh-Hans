# [!DNL Target] AEM集成可能会导致显示“工作区无效”的错误

## 描述


集成AEM和 [!DNL Target] 使用IMS时，在将XF从AEM传递到AEM的error.log文件中会看到一个错误，该错误显示“工作区无效” [!DNL Target].


## 分辨率


作为解决方法，请尝试使用旧版集成方式，并查看这是否解决了无效的工作区错误。
