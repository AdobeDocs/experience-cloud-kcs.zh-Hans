# Target AEM集成可能会导致显示“工作区无效”的错误

## 描述


使用IMS集成AEM和Target时，在将XF从AEM传递到Target时，AEM的error.log文件中会显示一个错误，表示“工作区无效”。


## 分辨率


作为解决方法，请尝试使用旧版集成方式，并查看这是否解决了无效的工作区错误。
