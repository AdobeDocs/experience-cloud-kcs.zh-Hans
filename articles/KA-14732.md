# [!UICONTROL Recommendations] 标准显示错误结果

## 描述


<b>环境</b>

Target



<b>问题/症状</b>

[!UICONTROL Recommendations] 的CSV文件。


## 分辨率


<b>原因</b>

由于的包含规则，可能会过滤结果 `primaryCategory`.



<b>Recommendations</b>

1. 查看活动定义和标准定义。
2. 通过CSV下载检查相应环境的结果。
3. 检查上述任何结果中是否包含包含规则。 如果没有，则会向访客显示备用推荐。
