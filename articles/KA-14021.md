# A4T活动中报表受众的入选体验不同

## 描述


<b>环境</b>

Target



<b>问题/症状</b>

A4T Target活动报表中报表受众的入选体验为何不同？




## 分辨率


- 全局入选者不同的原因在于，它是根据Target中捕获的默认/主要目标量度值来确定的，而报表入选者则完全是根据从Analytics获取的数据来确定的。
- 理想情况下，二者应相同，但鉴于数据可能有所不同，这两者可能不同。
