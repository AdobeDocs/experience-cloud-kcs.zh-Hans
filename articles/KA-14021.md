# A4T活动中报表受众的入选体验不同

## 描述


<b>环境</b>

Target



<b>问题/症状</b>

A4T中报表受众的入选体验为何不同 [!DNL Target] 活动报表？




## 分辨率


- 全局入选者不同的原因在于，它是根据 [!DNL Target]，而报告入选者则完全根据从 [!DNL Analytics].
- 理想情况下，二者应相同，但鉴于数据可能有所不同，这两者可能不同。
