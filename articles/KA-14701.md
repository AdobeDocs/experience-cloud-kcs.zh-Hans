# Adobe Campaign:错误：无法解析表达式&#39;[@registrationToken]&#39;

## 描述

已报告在尝试使用移动应用程序渠道时引发以下错误：

```
error XTK-170036 Unable to parse expression '@registrationToken'
```

## 分辨率


解决方案是安装 [!UICONTROL 移动设备应用程序渠道] 在执行实例上。
