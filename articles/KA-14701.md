# Adobe Campaign：错误：无法解析表达式&#39;[@registrationToken]’

## 描述

据报告，在尝试使用移动应用程序渠道时引发以下错误：

```
error XTK-170036 Unable to parse expression '@registrationToken'
```

## 解决方法


解决方法是安装 [!UICONTROL 移动应用程序渠道] 在执行实例上。
