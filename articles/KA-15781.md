# Campaign Classic — 描述性分析中未填充表

## 描述


当我们导出时 [!UICONTROL 描述性分析] 报表时，图表部分导出的图表不会。


## 分辨率


我们的工程团队已接受此作为增强请求，因为如果 [!UICONTROL Excel] 选项，并且图表部分正确导出。

此问题已在21.1.4版本（版本9343）及以后版本中修复。
