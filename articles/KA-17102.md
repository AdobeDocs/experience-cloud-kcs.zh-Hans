# 分类API导入错误关系0当前未设置为导入数据

## 描述


分类(SAINT)API中出现API导入错误“关系0当前未设置为导入数据”的原因是什么？


## 分辨率


将变量“product”与大写“Product”一起使用时，会看到错误“Relation 0 is not set to import data”（关系0当前未设置为导入数据）。 分类API将仅接受大小写为“product”且带有“p”的变量。
