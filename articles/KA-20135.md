# 阻止SQL注入攻击

## 描述


到底如何阻止SQL注入攻击？


## 分辨率


AEMas a Cloud Service不提供开箱即用的可配置IP阻止列表。 因此使用 [!DNL Apache] `mod_redirect` 或 [!DNL Dispatcher] 在项目级别。

SQL注入被设计阻止：默认的CQ设置既不包括也不需要传统数据库。 所有数据都存储在内容存储库(CRX)中。 所有访问权限仅限于经过身份验证的用户，且只能通过JCR API执行。 仅支持搜索查询(`SELECT`)。
