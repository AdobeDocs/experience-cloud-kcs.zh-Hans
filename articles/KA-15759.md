# 如何查找 [!DNL Launch] 暗旗？

## 描述

我如何知道我当前拥有的AdobeAnalysis Workspace功能？ 

## 分辨率

[!DNL Launch] 将暗标记设置为IMS组织级别。

登录 [!DNL Analytics]，然后转到 [!UICONTROL 工作区].

* [!UICONTROL 工作区] > [!UICONTROL 帮助] > [!UICONTROL 关于Workspace]
* 看主动特征标志
