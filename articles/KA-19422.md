# ACC - AEM集成 — 如果有少数图像未从AEM模板渲染到Adobe Campaign中

## 描述

 用例 — 如果有少数页面未从AEM模板渲染到Adobe Campaign中

## 分辨率


从AEM导入的模板可能会正确渲染图像。 要获得解决方案，请始终检查 [!UICONTROL 外部帐户] 分配了适当的权限。 请咨询您的AEM团队以验证相应的访问权限。
