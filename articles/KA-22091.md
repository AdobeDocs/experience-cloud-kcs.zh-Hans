# 管道错误：“缺少end_aem_createsegmentstore属性”


“当管道中发生错误时： `The end_aem_createsegmentstore property is missing`，按照本文中的说明解决问题。”

## 描述 {#description}


### <b>环境</b>

Experience Manager



### <b>问题/症状</b>

此 <b>持续集成</b> 和 <b>连续部署</b> <b>(CI/CD)</b> 管道在 *部署到开发* 暂存。

## 解决方法 {#resolution}


检查管道时，可以看到它因错误而失败 *错误代码*： *BLOCKED_QUEUE*.

这意味着复制队列被阻止，导致其他可变内容的安装受阻。

在日志中，错误 <b>`[` 1`]` </b> 在部署时间范围内存在。 很可能是节点 —  <b>`[` 2`]` </b> 在发布服务器和/或预览服务器上不存在，因此复制失败。

清除复制队列后，管道部署成功。
