# Adobe Campaign： API 504超时错误

## 描述 {#description}




### 问题/症状



- 对Campaign进行API调用时收到504错误






### 环境



- Apache RequestReadTimeout选项设置为20秒（默认设置）



## 解决方法 {#resolution}




### 解决/解决方法的步骤



- 增加此超时以与Amazon ELB（60秒）匹配，因为某些API调用可能会超过默认的20秒并超时。
