# 未知身份命名空间

## 描述 {#description}

为什么存在我们未在 AEP Identities UI 中创建的非标准身份命名空间？

## 解决方法 {#resolution}


您看到的可能是创建了相关“AEP 身份命名空间”的 Audience Manager 跨设备数据源。不确定“AEP 身份命名空间”和“AAM 跨设备数据源”是如何连接的？ 请参阅[这篇文章](https://experienceleague.adobe.com/docs/experience-cloud-kcs/kbarticles/KA-21305.html)
