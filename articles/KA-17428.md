# 管理对status.adobe.com的订购

## 描述 {#description}

<b>环境</b>
- Experience Cloud



<b>问题/症状</b>
管理订阅或订阅status.adobe.com通知的步骤。


## 解决方法 {#resolution}


1. 转到status.adobe.com
2. 使用Adobe ID登录
3. 单击“管理订阅”
4. 单击“创建订阅”
5. 在“Experience Cloud”下 — 选择要订阅的解决方案和子组件 — 继续
6. 选择区域和事件类型 — 继续
7. 确认订阅，然后单击完成
