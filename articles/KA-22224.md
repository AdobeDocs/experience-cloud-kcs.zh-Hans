# 未生成Dynamic Media智能裁剪

## 描述 {#description}


### 环境

Dynamic Media Classic

### 问题/症状

在应生成Dynamic Media SmartRancs的文件夹中，不会生成这些内容（图像配置文件将覆盖这些内容）。


## 解决方法 {#resolution}


这是由于缺少所致 `jcr:content` 在树中的文件夹节点上。 您可以通过从AEM UI中为Target文件夹打开属性页来放置这些项。 然后单击 <b>保存并关闭</b> 树中所有中间文件夹的位置。
