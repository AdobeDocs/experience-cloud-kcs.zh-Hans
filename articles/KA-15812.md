# 扩充/查询活动：“所有行”选项会给出错误

## 描述


<b>环境</b>
Adobe Campaign Classic

<b>问题</b>
使用 <b>所有行</b> 扩充或查询活动的选项会导致错误：


```
BAS-010000 Unable to complete the operation in current status.
```



## 分辨率


<b>解决方案</b>
的 <b>所有行</b> 选项在单击 <b>完成</b> ( <b>下一个</b>)，然后运行工作流。

UI似乎存在一个小问题， <b>下一个</b> 按钮 <b>所有行</b> 选项。
