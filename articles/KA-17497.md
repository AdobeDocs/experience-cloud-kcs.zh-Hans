# 了解AEM Dispatcher缓存哪些请求

## 描述

AEM Dispatcher缓存哪些HTTP请求和响应？ (AEM Dispatcher 4.3.3)<br><br><br>

## 分辨率


本文档介绍了影响通过AEM Dispatcher模块的请求和响应的缓存能力的最重要方案。 未涵盖所有缓存规则。
<br><br><br><br> <br><br>HTTP请求和缓存规则：<br><br><br><br> <br><br>
要使调度程序可缓存HTTP请求，它必须遵循以下规则。
<br><br><br><br> <br><br>1. URL是绝对的（不以开头）。 或……)具有文件扩展名，且URL中的文件名可用作文件系统上的文件。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `URI not canonical: %s.``Unable to map URI to ``file``: %s.``cache ``file` `path too long``temporary ``file` `path too long``request URL has no extension` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| /content/test | 未缓存 |
| --- | --- |
| /content/test.html | cached |

<br><br><br><br> <br><br>2. 在第一个文件扩展名后没有斜杠。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `request URL has trailing slash` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| /content/test.html/ | 未缓存 |
| --- | --- |
| /content/test.html/foo.jpg | 未缓存 |
| /content/test.html | 缓存 |

<br><br><br><br> <br><br>3. 它使用HTTPGET或HEAD方法。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `request method is neither GET nor HEAD` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HEAD/content/test.html HTTP/1.1 | 缓存 |
| --- | --- |
| GET/content/dam/test.jpg HTTP/1.1 | 未缓存 |
| POST/content/test.html HTTP/1.1 | 未缓存 |

<br><br><br><br> <br><br>4. DISPATCHER .ANY场配置中定义的/CACHE = /RULES拒绝该请求。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `URI not ``in` `cache rules: %s``request URL not ``in` `cache rules` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| 场具有/cache = /rules ，并且收到 <b>/content/test.html</b><br><br>/cache<br><br>{<br><br>...<br><br>/rules<br><br>{<br><br>/0001 {<br><br>类型&quot;deny&quot;<br><br>glob &quot;\*&quot;<br><br>}<br><br>} | 未缓存 |
| --- | --- |
| 场具有/cache = /rules ，并且收到 <b>/content/test.html</b><br><br>/cache<br><br>{<br><br>...<br><br>/rules<br><br>{<br><br>/0001 {<br><br>类型“allow”<br><br>glob &quot;\*&quot;<br><br>}<br><br>} | 缓存 |

<br><br><br><br> <br><br>5. HTTP请求不包含授权标头，或者包含标头，但场配置中的/ALLOWAUTHORIZED设置为1。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `request contains authorization` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HTTP请求：<br><br>GET/content/test.html HTTP/1.1<br>  授权：基本YWxhZGRpbjpvcGVuc2VzYW1l<br><br>dispatcher.any场/cache = /allowAuthorized配置：<br><br>/cache {<br>    /allowAuthorized &quot;0&quot;<br>  } | 未缓存 |
| --- | --- |
| HTTP请求：<br><br>GET/content/test.html HTTP/1.1<br>  授权：基本YWxhZGRpbjpvcGVuc2VzYW1l<br><br>dispatcher.any场/cache = /allowAuthorized配置：<br><br>/cache {<br>    /allowAuthorized &quot;1&quot;<br>  } | 缓存 |

<br><br><br><br> <br><br>6. HTTP请求在COOKIE标头中包含登录令牌或授权COOKIE，或者它包含（或同时包含）这些COOKIE中的任一COOKIE，并且场配置中的/ALLOWAUTHORIZED设置为1。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `request contains authorization` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HTTP请求：<br><br>GET/content/test.html HTTP/1.1<br>  Cookie:login-token=...<br><br>dispatcher.any场/cache = /allowAuthorized配置：<br><br>/cache {<br>    /allowAuthorized &quot;0&quot;<br>  } | 未缓存 |
| --- | --- |
| HTTP请求：<br><br>GET/content/test.html HTTP/1.1<br>  Cookie:authorization=YWxhZGRpbjpvcGVuc2VzYW1l<br><br>dispatcher.any场/cache = /allowAuthorized配置：<br><br>/cache {<br>    /allowAuthorized &quot;0&quot;<br>  } | 未缓存 |
| HTTP请求：<br><br>GET/content/test.html HTTP/1.1<br>  Cookie:login-token=...<br><br>dispatcher.any场/cache = /allowAuthorized配置：<br><br>/cache {<br>    /allowAuthorized &quot;1&quot;<br>  } | 缓存 |

<br><br><br><br> <br><br>7. URL中有一个QUERYSTRING参数，允许通过场的/IGNOREURLPARAMS配置忽略该参数。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `request contains a query string` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HTTP请求：<br><br>GET/content/test.html?test=1 HTTP/1.1<br><br>场/缓存配置：<br><br>/ignoreUrlParams {<br>     /0001 { /type &quot;allow&quot; /glob &quot;\*&quot; } <br>  } | 缓存 |
| --- | --- |
| HTTP请求：<br><br>GET/content/test.html?test=1 HTTP/1.1<br><br>场/缓存配置：<br><br>/ignoreUrlParams {<br>     /0001 { /type &quot;deny&quot; /glob &quot;\*&quot; } <br>     /0001 { /type &quot;allow&quot; /glob &quot;test&quot; }<br>  } | 缓存 |
| HTTP请求：<br><br>GET/content/test.html?test=1 HTTP/1.1<br><br>dispatcher.any场/cache = /allowAuthorized配置：<br><br>/ignoreUrlParams {<br>     /0001 { /type &quot;deny&quot; /glob &quot;\*&quot; } <br>     /0001 { /type &quot;allow&quot; /glob &quot;q&quot; }<br>  } | 未缓存 |

<br><br><br><br> <br><br>HTTP响应和缓存规则：<br><br><br><br> <br><br>
如果满足以下条件，则可缓存从AEM返回的HTTP响应。
<br><br><br><br> <br><br>1. 调度程序能够发送并接收来自定义的“呈现”之一的200 OK响应。<br><br><br><br><br><br>注释：

如果否 [/timeout](https://helpx.adobe.com/experience-manager/dispatcher/using/dispatcher-configuration.html#DefiningPageRenderersrenders) 在/renders中设置，或将其设置为 <b>0</b> 那么，即使实例关闭，它也会一直等待与AEM实例的连接。


<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `Unable to send request to remote server.``Unable to receive response from remote server.``Remote server returned: %s``No backend available.` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HTTP请求：<br><br>200 OK | 缓存 |
| --- | --- |
| HTTP请求：<br><br>404页面未找到 | 未缓存 |

<br><br><br><br> <br><br>2. 响应中不存在以下响应标头：<br><br><br><br> <br><br>
- <b>调度程序：无缓存</b>
- <b>缓存控制：无缓存</b>
- <b>Pragma:无缓存</b>

<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `Backend forbids caching: %s, sent: %s"``response contains no_cache` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HTTP响应：<br><br>200 OK<br>  调度程序：无缓存 | 未缓存 |
| --- | --- |
| HTTP响应：<br><br>200 OK<br>  缓存控制：无缓存 | 未缓存 |
| HTTP响应：<br><br>200 OK<br>  Pragma:无缓存 | 未缓存 |
| HTTP响应：<br><br>200 OK | 缓存 |

<br><br><br><br> <br><br>3. CONTENT-LENGTH值大于零字节。<br><br><br><br> <br><br>
<b>相关日志消息：</b>
<br><br><br><br><br> <br><br> <br><br><br><br>

| `response content length is zero` |
| --- |

<br><br><br><br><br> <br><br>
<b>示例：</b>
<br><br><br><br> <br><br>

| HTTP响应：<br><br>200 OK<br>  Content-Length:0 | 未缓存 |
| --- | --- |
| HTTP响应：<br><br>200 OK<br>  Content-Length:3120 | 缓存 |
